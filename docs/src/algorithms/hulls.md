## Graham's scan

```@docs
GrahamScan
```

```@example hull
using Meshes, MeshViz
import CairoMakie as Mke

pset = PointSet(rand(Point2, 100))
chul = hull(pset, GrahamScan())

fig = Mke.Figure(resolution = (800, 400))
viz(fig[1,1], chul)
viz!(fig[1,1], pset, color = :black)
fig
```
