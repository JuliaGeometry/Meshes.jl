## Laplacian

```@docs
laplacematrix
```

```@example matrices
grid = CartesianGrid(10, 10)

laplacematrix(grid, kind = :uniform)
```

```@example matrices
points = [(0, 0), (1, 0), (0, 1), (1, 1), (0.5, 0.5)]
connec = connect.([(1, 2, 5), (2, 4, 5), (4, 3, 5), (3, 1, 5)])
mesh = SimpleMesh(points, connec)

laplacematrix(mesh, kind = :cotangent)
```
